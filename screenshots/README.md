# Screenshots Folder

This folder contains screenshots demonstrating the AI Personal Assistant in action.

## Recommended Screenshots

Add the following screenshots to this folder:

1. **telegram-demo.png** - Example of bot conversation showing task extraction
2. **notion-database.png** - Screenshot of Notion database with sample tasks
3. **calendar-integration.png** - Google Calendar view with scheduled tasks
4. **n8n-workflow.png** - Overview of n8n workflow structure
5. **confirmation-message.png** - Bot confirmation message example

## How to Add Screenshots

1. Take screenshots on your device
2. Save them with descriptive names
3. Place them in this folder
4. Reference them in documentation with relative paths:
   ```markdown
   ![Description](../screenshots/filename.png)
   ```

## Image Guidelines

- **Format:** PNG or JPG
- **Size:** Optimize for web (< 1MB per image)
- **Resolution:** High enough to read text clearly
- **Privacy:** Remove sensitive information (names, emails, personal data)
